PLease Install and run mongoDb before trying to run the project
Open the project folder in an IDE and open terminal at that path
Install node 
Install all the required packages 'Express' 'Body Parser' 'Mongoose' 'Express Handlebars' 'nodemon'
Connect to the mongoDb database, a schema called User will be created in your local database
Run the project by using this commmand nodemon 'server.js'
After running the project check the terminal console for 2 success messages
 	-Express Server started at port 3000
	-MongoDB connected successfully
In your browser go the following link to access the application 
	-http://localhost:3000/user
